Vanta.js Birds Animation

A beautiful 3D bird flocking simulation that creates organic motion patterns. Perfect for background animations that need a natural, flowing feel.

// Initialize with these options:
VANTA.BIRDS({
  el: "#your-element",
  backgroundColor: 0x1f1d1d,
  color1: 0x6e44ff,
  color2: 0xaa48ec,
  birdSize: 1.2,
  quantity: 4
});

Particles.js Animation

Interactive particle system with physics-based movement and connections. Great for creating dynamic, engaging backgrounds with user interaction.

particlesJS('particles-js', {
  particles: {
    number: { value: 100 },
    color: { value: ["#ff6666", "#6e44ff"] },
    shape: { type: "circle" },
    line_linked: {
      enable: true,
      distance: 150,
      color: "#6811a2"
    }
  }
});

Vanta.js Net Animation

A mesmerizing 3D net-like structure that responds to mouse movement. Ideal for tech or network-related websites with a modern aesthetic.

VANTA.NET({
  el: "#your-element",
  color: 0x6e44ff,
  backgroundColor: 0x1f1d1d,
  points: 12,
  maxDistance: 20,
  spacing: 15
});

Vanta.js Waves Animation

A soothing wave simulation that creates calming, organic motion. Perfect for meditation, wellness, or ocean-related websites.

VANTA.WAVES({
  el: "#your-element",
  color: 0x6e44ff,
  shininess: 50,
  waveHeight: 20,
  waveSpeed: 0.5,
  zoom: 0.8
});

GSAP Animations

Powerful timeline-based animations for UI elements, scroll effects, and complex sequences. The industry standard for high-performance web animations.

// Basic animation
gsap.to(".element", {
  duration: 1,
  x: 100,
  rotation: 360,
  ease: "power2.out"
});

// Stagger animation
gsap.from(".items", {
  duration: 0.5,
  opacity: 0,
  y: 50,
  stagger: 0.1
});
